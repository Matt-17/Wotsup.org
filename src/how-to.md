## How to Use Wotsup.org

Note: the on-site search box has been removed. Use the alternatives below to find format information.

Finding information

- Browse by category using the site navigation to the left or the header menu. Categories group similar formats (archive, graphics, audio, etc.).
- Use external search engines with a site filter, for example: `site:wotsit.org zip format` or `site:wotsup.org "ZIP"`.
- If you know a file extension, try browsing the relevant category or lookup pages (e.g., Archive / Compression formats).

Format pages include a short description, author/attribution (when known), and links to available specification documents or downloads.

Downloading files

- Click the file link on a format page to download a specification or resource. Files are served from the `assets/files/` area of the site.

Reporting problems or missing info

- Found a broken link, error, or missing reference? Open an issue on GitHub: `https://github.com/Matt-17/Wotsup.org/issues`.
- Prefer to suggest a change first? Open an issue describing the change and the source material.

Contributing

- Want to contribute content or fixes? See our [Contributing guide](/contributing/) for a quick, friendly introduction and the best way to get started.

Need help

- Check the [FAQ](/faq/) for common questions.
- For other inquiries, use the contact links on the site footer.
