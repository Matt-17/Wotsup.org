This site exists to provide a useful resource library for programmers. If you have any complaints regarding the site or the content then please contact complaint@wotsup.org.

For faster handling, please provide:

- The page or URL affected
- A brief description of the complaint
- Any supporting screenshots or examples

If the complaint relates to licensing or removal requests, include the rights-holder contact details so we can verify and act promptly.
